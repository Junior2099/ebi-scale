# Escala Mensal - EBI

Pequena aplicação em HTML/CSS/JavaScript para gerar uma escala mensal com os dias e horários configurados:

- Dias considerados: domingo, segunda, quarta e sexta-feira.
- Horários: domingo 08:00 e 18:00; segunda/quarta/sexta 20:00.

Como usar:

1. Abrir `index.html` no navegador.
2. Cadastrar as tias no painel à esquerda.
3. Selecionar o mês (padrão: fevereiro de 2026).
4. Clicar em um horário para atribuir uma tia.

Persistência: as tias e as atribuições são salvas no `localStorage` do navegador.
